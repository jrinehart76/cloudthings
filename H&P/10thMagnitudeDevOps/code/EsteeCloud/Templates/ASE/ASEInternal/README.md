# Azure App Service Plans
This template deploys an App Service Environment with an App Service Plan. The Isolated pricing tier of the App Service Plan is only available to be used with App Service Environments that will host them and provides non-shared infrastructure that is reserved for your solution.
This pricing tier provides advanced netowrking option and scale
# Notes
Production pricing tiers are only available to plans that are not hosted within an App Service environment
Dev/Test pricing tiers are only available to plans that are not hosted within an App Service environment
