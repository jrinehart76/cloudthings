## Wiki Functionality
The best way to edit this wiki is through the editor in VSTS. For more info, refer to [this doc](https://docs.microsoft.com/en-us/vsts/collaborate/add-edit-wiki).

For help on Markdown syntax, refer to [this document](https://guides.github.com/features/mastering-markdown/)

## Git Repo
Under the hood, this wiki is just a Git repo. You can access the repo directly by navigating to [this link](https://elektacloud.visualstudio.com/ECS%20Azure%20Demo%20v2/_git/ECS%20Azure%20Demo%20v2.wiki/). You can even pull the repo and edit offline.
